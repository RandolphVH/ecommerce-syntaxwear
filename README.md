# SyntaxWear

SyntaxWear é um projeto de landing page para uma loja virtual de tênis e sneakers, desenvolvido com HTML5 e CSS3. O objetivo é demonstrar um layout moderno, responsivo e visualmente atrativo para um e-commerce, com foco em experiência de navegação tanto em desktop quanto em dispositivos móveis.

## 📌 Sobre o projeto

Este projeto simula um site de comércio eletrônico com:
- cabeçalho com navegação e ícones
- seção hero com destaque de campanha
- cards de categorias
- grade de produtos
- newsletter e footer
- menu hambúrguer responsivo para telas menores

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- Arquitetura de estilos organizada por componentes

## ▶️ Como visualizar o site

Você pode abrir o arquivo [index.html](index.html) diretamente no navegador, ou usar um servidor local simples para evitar problemas com recursos estáticos.

### Opção 1: abrir diretamente
1. Entre na pasta do projeto
2. Abra o arquivo [index.html](index.html) no navegador

### Opção 2: usar um servidor local
No terminal, dentro da pasta do projeto, execute:

```bash
python -m http.server 8000
```

Depois acesse:

```text
http://localhost:8000
```

## 🎨 Personalização

Você pode editar os seguintes arquivos para alterar o visual e o conteúdo do site:

- [index.html](index.html): texto, links e estrutura das seções
- [css/variables.css](css/variables.css): cores, fontes e tokens visuais
- [css/components/](css/components/): estilos dos componentes do layout
- [css/responsive.css](css/responsive.css): ajustes para telas menores
- [images/](images/): substituição de imagens e ícones
